<h1>Set Buillder Notation</h1> 
<div style="">
    <ul style="">
        <li>N = Natural Numbers</li>
        <li>What follows the : is a condition</li>
        <li>{0, 2, 4, ...} = {x ∈ N : ∃n ∈ N(x  2n)} = {x ∈ N : x is even}</li>
        <li>Intersection creates a new set of only common elements</li>
        <li>Subset means every element of A is in B, but A is not equal to B</li>
        <li>Subset underline means every element of A is in B</li>
        <li>Union creates a new set that combines two sets and leaves no repeating elements</li>
    </ul>
